# 网络

# 1. 七层网络协议

  • # 物理层

    集线器、中继器，传输比特流

  • # 数据链路层

    网桥、交换机，传输数据帧

  • # 网络层

    路由器，传输IP数据报；IP,ICMP,ARP

  • # 传输层

    网关；TCP,UDP

  • # 会话层

    管理主机之间的会话进程

  • # 表示层

    数据的加密、压缩、格式转换等

  • # 应用层

    FTP,Telnet,DNS,SMTP,HTTP
